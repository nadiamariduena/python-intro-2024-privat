## DATA types

<br>

### STR (string)

```python
# String data type
# literal assignment
first = 'Rick'
last = "Blaz"

print(type(first))
```

#### result

- once you run the code , you will see on the terminal that you got the **str** for string

```python
<class 'str'>
```

## BOOL (boolean)

```javascript
# String data type
# literal assignment
first = True
last = False

print(type(first))
// result
```
